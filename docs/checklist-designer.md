# Designer Checklist

## Bear Mascot
- [ ] 6 emotion states: thinking, curious, concerned, alert, encouraging, happy
- [ ] Small size (~80-100px), lives near the mic or bottom of screen
- [ ] Must be doable in <2 hours — keep it simple

## Key UI Components
- [ ] Burnout gauge (0-100 score, 3 levels: low/moderate/high)
- [ ] Health data cards (metric name + value + trend direction)
- [ ] Protocol card (3 daily actions)
- [ ] Berries reward animation
- [ ] Mic button (idle / recording / processing states)
- [ ] Nudge notification card
- [ ] Patient avatars (2-3 distinct profiles)

## Guidelines
- [ ] Dark background (projector readability)
- [ ] Mistral orange (#ff7000) as accent — rest is your call
- [ ] Must work fullscreen on projector (16:9, readable at 3m)
- [ ] Medical-tech feel — not too playful, not too clinical
