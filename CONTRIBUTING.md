# Contributing

Thanks for wanting to contribute — seriously. Star the repo, open an issue if something bugs you, and stay tuned. It's coming.

> **Contributions are not open yet.** The project is in early POC stage and iterating fast — the codebase changes daily. Opening contributions now would create more friction than value.

If you're interested in contributing, star the repo and watch for updates. Once the architecture stabilizes, we'll open issues and accept PRs.
